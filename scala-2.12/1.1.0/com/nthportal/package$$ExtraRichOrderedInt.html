  7.  def thenCompare[A](a1: A, a2: A)(implicit ord: Ordering[A]): Int

    Returns the result of the immediately previous comparison if it was not 0; otherwise the result of comparing two other things (a1 and a2).

    Returns the result of the immediately previous comparison if it was not 0; otherwise the result of comparing two other things (a1 and a2).

    This method is intended to be used to chain comparisons, as in the following example:

    case class PlayingCard(rank: Rank, suit: Suit) extends Ordered[PlayingCard] {
  override def compare(other: PlayingCard): Int = {
    (this.rank compare that.rank)
      .thenCompare(this.suit, that.suit)
  }
}

    While this method can called on arbitrary Ints, to be meaningful, it should only be called on the result of a previous comparison.
